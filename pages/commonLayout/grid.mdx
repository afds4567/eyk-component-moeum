# Grid

## Usage

CSS Grid를 사용하여 자식 컴포넌트들을 격자 형태로 배열하는 컴포넌트

```tsx filename="grid.tsx" {1} copy
<Grid rows={2} columns={2} gap={10}>
  <div>RED</div>
  <div>YELLOW</div>
  <div>GREEN</div>
  <div>BLUE</div>
</Grid>
```

## Component

import { Grid } from "../../components/Grid.tsx";

<Grid rows={2} columns={2} gap={10}>
  <div style={{ background: "red" }}>RED</div>
  <div style={{ background: "yellow" }}>YELLOW</div>
  <div style={{ background: "green" }}>GREEN</div>
  <div style={{ background: "blue" }}>BLUE</div>
</Grid>
